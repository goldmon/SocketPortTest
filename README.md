# 端口测试工具
- 这是一个基于Go的端口测试工具,CS架构
- 参数:server/client 端口 IP(服务端忽略)
- 固定端口通讯，客户端请求端口，服务端监听端口，客户端连接端口
- 用于某些企业网防火墙屏蔽端口探测

## 编译：
- go build .

## 服务端运行：
- main 软件执行文件
- server 服务器模式
- 5939 服务器监听端口
- 空格分割
```
main server 5939
```
## 客户端运行：
- main 软件执行文件
- client 客户端模式
- 5939 服务器端口
- 127.0.0.1 服务端IP
```
main client 5939 127.0.0.1
```
